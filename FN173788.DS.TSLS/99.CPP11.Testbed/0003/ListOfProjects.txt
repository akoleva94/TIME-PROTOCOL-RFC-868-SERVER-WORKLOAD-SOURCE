0003.0101 +++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
Project Name
	InfixToPostfixConvertor
Description
	Another branch of the Root Project, based on Reverse Polish Notation (Postfix Expressions Notation).
	Digits entered separately form value.
	Operation (+, -, *, /) add the value to the infix expression. The so formed infix expression is converted to postfix.
	After pressing of = postfix expression is evaluated and result is displayed in the value field.
	The digit entered after = clears both expressions and the iteration starts from the beginning.
Root Project
